Features

Everything a human does.
Nothing a bot does.

Every setting is built to mirror real human typing behaviour — down to how your fingers slow at the edges of words.

⚡

Adjustable WPM

Set any words-per-minute speed. From a slow, careful typist to a fast touch-typist — fully in your control.

🎯

Realistic Typos

Three typo styles — random letters, adjacent keys on QWERTY, and transposed characters. Looks exactly like real mistakes.

⌨️

Capitalization Errors

Occasionally types a letter in the wrong case then self-corrects — like accidentally holding Shift a beat too long.

📉

Fatigue Simulation

Typing gradually slows down over the course of the text, like a real person getting tired as they go.

🌊

Word Acceleration

Slightly slower at the start and end of each word, fastest mid-word — matches the natural rhythm of real fingers.

⏸️

Human Breaks

Randomly pauses mid-text like someone reading ahead, and always hesitates after sentences and punctuation.

🧩

Chunk Mode

Pauses every few words and waits for a hotkey to continue — perfect for copying text piece by piece.

🔄

Loop / Repeat

Type the same text multiple times with a customizable delay between each run. Set it and walk away.

🪟

Background Mode

Types into any window — even when the app isn't focused. Lock onto your target window and switch freely.

🎛️

Presets

Save your favourite settings as named presets and switch between them instantly. Never reconfigure from scratch.

📂

Load from File

Import any .txt file directly into the app. Perfect for long documents, scripts, or essays.

🤖

Claude Code Integration

Send text straight from Claude Code with one curl command. AI writes it, Human Typer types it.

Getting Started

Up and running
in three steps.

1

Clone & install

Clone the repo and install the dependencies with pip.

git clone https://github.com/The-Bean-Beater/human-typer.git cd human-typer pip install -r requirements.txt
2

Run the app

Launch Human Typer from your terminal.

python3 human_typer.py
3

Paste, configure, type

Drop in your text, dial in your settings, click Start — or use a hotkey.

Global Hotkeys

Work even when the app is hidden in the background.

Ctrl + Opt + HStart typing
Ctrl + Opt + SStop typing
Ctrl + Opt + SpaceResume chunk
AI Integration

Built for Claude Code.

Ask Claude to generate anything — an email, an essay, a cover letter — then send it straight to Human Typer with one command. It fills the box and starts typing automatically. No copy-paste needed.

Get Started
Fill the text box
curl -X POST http://localhost:7799/type \ -d "your text here"
Fill + start typing immediately
curl -X POST http://localhost:7799/type-and-start \ -d "your text here"
