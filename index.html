Features

Everything a human does.
Nothing a bot does.

Every setting is designed to mirror real human typing behavior.

⚡

Adjustable WPM

Set any words-per-minute speed. From a slow careful typist to a fast touch-typist — fully in your control.

🎯

Realistic Typos

Three typo styles — random letters, adjacent keys on QWERTY, and transposed characters. Looks exactly like real mistakes.

⌨️

Capitalization Errors

Occasionally types a letter in the wrong case then self-corrects — like accidentally holding Shift a beat too long.

📉

Fatigue Simulation

Typing gradually slows down over the course of the text, like a real person getting tired as they go.

🌊

Word Acceleration

Slightly slower at the start and end of each word, fastest mid-word — matches the natural rhythm of real fingers.

⏸️

Human Breaks

Randomly pauses mid-text like someone reading ahead, and always hesitates after sentences and punctuation.

🧩

Chunk Mode

Pauses every few words and waits for you to press a hotkey to continue — perfect for copying text piece by piece.

🔄

Loop / Repeat

Type the same text multiple times with a customizable delay between each run. Set it and walk away.

🪟

Background Mode

Types into any window — even when the app isn't focused. Lock onto your target window and switch freely.

🎛️

Presets

Save your favorite settings as named presets and switch between them instantly. Never reconfigure from scratch.

📂

Load from File

Import any .txt file directly into the app. Perfect for long documents, scripts, or essays.

🤖

Claude Code Integration

Send text directly from Claude Code with a single curl command. Let AI generate content and Human Typer type it out.

AI Integration

Built for Claude Code.

Ask Claude to generate anything — an email, an essay, a message — then send it straight to Human Typer with one command. It fills the box and starts typing automatically.

Get Started
Fill the text box
curl -X POST http://localhost:7799/type \ -d "your text here"
Fill + start typing instantly
curl -X POST http://localhost:7799/type-and-start \ -d "your text here"
Installation

Up and running
in seconds.

Three commands and you're good to go.

1

Clone the repo

git clone https://github.com/The-Bean-Beater/human-typer.git cd human-typer
2

Install dependencies

pip install -r requirements.txt
3

Run the app

python3 human_typer.py
Hotkeys

Hands-free control.

Global hotkeys work even when the app is in the background.

Ctrl + Opt + H Start typing
Ctrl + Opt + S Stop typing
Ctrl + Opt + Space Resume chunk
