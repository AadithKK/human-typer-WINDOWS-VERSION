Features

It doesn't just type fast.
It types human.

Every setting targets a specific tell that gives bots away.

🎯

Three kinds of typos

Wrong letter, adjacent key on QWERTY, or transposed swap — all with realistic backspace-and-correct behaviour. Set intensity from 0% to 100%.

12
humanization settings
📉

Fatigue simulation

Gradually slows down over the text like a real person settling in — or wearing out.

🌊

Word acceleration

Fingers are slowest at the edges of words and fastest mid-word. Human Typer models this exactly.

⏸️

Human breaks

Random pauses mid-text, longer hesitations after punctuation.

🧩

Chunk mode

Pause every few words, resume with a hotkey — type section by section.

🔄

Loop / repeat

Type the same text multiple times, with a delay between runs.

🪟

Background mode

Locks onto any window at start. Types into it even when Human Typer isn't focused — switch apps freely.

⌨️

Capitalization errors

Occasionally holds Shift a beat too long, types the wrong case, catches itself. Subtle but effective.

∞
Free forever
🎛️

Presets + history

Save named setting profiles and reload them in one click. Text history keeps your last 10 typed entries — load any of them back instantly.

Installation

Three commands.

That's all it takes.

1

Clone the repo

git clone https://github.com/The-Bean-Beater/human-typer.git cd human-typer
2

Install dependencies

pip install -r requirements.txt
3

Run it

python3 human_typer.py
Global Hotkeys

Works when the window's closed.

All three hotkeys fire even when Human Typer is hidden in the menu bar or behind other windows.

Ctrl + Opt + HStart typing
Ctrl + Opt + SStop typing
Ctrl + Opt + SpaceResume chunk
AI Integration

Claude writes it.

Ask Claude Code to generate any text, then fire one curl command to send it straight here. Human Typer fills the box and starts typing automatically — no copy-paste, no switching windows.
